# VooStack Homebrew Tap

This repository contains Homebrew formulae for VooStack tools.

## Installation

```bash
brew tap voostack/tap
brew install flightstack
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `flightstack` | CLI for FlightStack - Self-hosted CI/CD for Flutter |

## Updating

```bash
brew update
brew upgrade flightstack
```

## Uninstalling

```bash
brew uninstall flightstack
brew untap voostack/tap
```
